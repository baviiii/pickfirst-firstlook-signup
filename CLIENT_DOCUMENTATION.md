## Getting Started

### Creating an Account
1. Click "Sign Up" on the homepage
2. Choose "I'm a Buyer"
3. Enter your email and create a password
4. Verify your email address
5. Complete your profile with contact information

### Logging In
1. Click "Log In" on the homepage
2. Enter your email and password
3. Click "Log In"
   - Forgot password? Click "Reset Password"

## Account Management

### Profile Setup
- Go to "My Profile" from the main menu
- Upload a profile picture
- Add/update your contact information
- Set your notification preferences

### Password Management
- Change password: Go to Account Settings → Security
- Reset password: Use "Forgot Password" on login page
- Enable two-factor authentication for added security

## Property Search & Browsing

### Basic Search
1. Use the search bar at the top of any page
2. Enter a location, address, or suburb
3. Filter results using the sidebar options
   - Price range
   - Property type
   - Bedrooms/Bathrooms
   - Square footage
   - Features (pool, garage, etc.)

### Advanced Search
1. Click "Advanced Search" below the main search bar
2. Set detailed search criteria
   - Property features
   - Land size
   - Year built
   - Parking spaces
   - And more...

### Property Details
View comprehensive information including:
- High-resolution photo gallery
- Virtual tour (if available)
- Floor plans
- Property features
- Location map
- Nearby schools and amenities
- Price history
- Estimated mortgage calculator

## Saved Properties & Favorites

### Saving Properties
1. Click the heart icon (♥) on any property card
2. Access saved properties from "My Favorites" in your account

### Organizing Favorites
- Create custom collections (e.g., "Family Homes", "Investment Properties")
- Add notes to saved properties
- Share collections with family or advisors

## Property Alerts

### Setting Up Alerts
1. Go to "Property Alerts" in your account
2. Click "Create New Alert"
3. Set your search criteria
4. Choose notification frequency (instant, daily, weekly)

### Managing Alerts
- Edit existing alerts
- Pause/resume alerts
- Delete alerts you no longer need
- View alert history

## Making Inquiries

### Contacting Agents
1. On any property page, click "Contact Agent"
2. Complete the inquiry form
3. Your message will be sent directly to the listing agent
4. Continue the conversation via your Messages

### Inquiry Status
Track the status of your inquiries:
- New: Agent hasn't responded yet
- In Progress: Agent has responded
- Closed: Inquiry resolved

## Scheduling Viewings

### Requesting a Viewing
1. On a property page, click "Schedule Viewing"
2. Select an available time slot from the agent's calendar
3. Add any special requests or questions
4. Confirm your appointment

### Managing Appointments
- View all upcoming viewings in your dashboard
- Reschedule or cancel appointments (with 24 hours notice)
- Receive email/SMS reminders
- Add appointments to your personal calendar

## Messages & Communication

### Inbox
- Access all your messages in one place
- Organized by property/conversation
- Read receipts show when messages are viewed

### Notifications
- Choose how you want to be notified:
  - Email
  - Push notifications
  - SMS (premium feature)
- Set quiet hours for notifications

- Priority customer support

3. Choose payment method
4. Confirm your subscription

## Troubleshooting

### Common Issues
- **Can't log in?** Try resetting your password
- **Not receiving emails?** Check spam folder and notification settings
- **App not working?** Try updating to the latest version
- **Slow performance?** Clear your browser cache or try a different browser

### Browser Compatibility
- Chrome (latest version)
- Safari (latest version)
- Firefox (latest version)
- Edge (latest version)
